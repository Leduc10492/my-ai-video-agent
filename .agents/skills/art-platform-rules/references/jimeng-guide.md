# 即梦 (Jimeng) 提示词指南

## 语言特点

即梦使用中文提示词，偏好诗意化、意境化的描述方式。

## 结构模板

```
[主体] + [细节] + [环境/氛围] + [艺术风格] + [技术参数]
```

## 主体描述

### 人物描述
```
一位年约二十岁的少女
身着青色长裙的女子
白发苍苍的老者
英姿飒爽的年轻将军
```

### 动作描述
```
静立于/伫立于
轻抚着/手持着
凝视远方/回眸一笑
漫步在/奔跑于
```

## 环境描述

### 自然场景
```
远处是层叠的青山
薄雾缭绕的湖畔
落日余晖中的田野
繁星点点的夜空下
```

### 建筑场景
```
古朴的石桥上
飞檐翘角的亭台楼阁
斑驳的城墙边
灯火通明的街市
```

## 氛围词汇

### 情感氛围
```
意境悠远、宁静祥和
孤寂萧索、热烈奔放
神秘莫测、温馨浪漫
```

### 时间氛围
```
晨曦微露、正午阳光
黄昏暮色、月明星稀
```

## 艺术风格

### 中国传统风格
```
工笔画风格
写意水墨风格
青绿山水风格
敦煌壁画风格
```

### 现代风格

```
国潮插画风格

写实摄影风格
```

## 技术参数

```
高清细节
4K画质

```

## 完整示例

### 示例1: 古典人物
```
一位身着青色长裙的少女静立于古桥之上，
远处是层叠的青山，薄雾缭绕，溪水潺潺，
意境悠远，工笔画风格，高清细节，4K画质
```

### 示例2: 动态场景
```
英姿飒爽的年轻将军策马扬鞭，
尘土飞扬，战旗猎猎，夕阳如血，
热血沸腾，国潮插画风格，超高清晰度
```

### 示例3: 现代风格
```
短发少女坐在咖啡馆窗边，阳光透过玻璃洒落，
手中捧着一本书，神情专注而安详，
温馨浪漫，写实摄影风格，精细渲染
```

## 避免事项

❌ 使用英文词汇
❌ 过于直白的描述
❌ 参数化表达(--ar等)
❌ 技术性词汇过多

## 优化技巧

1. **层次感**: 由近及远描述场景
2. **意境化**: 使用四字成语增加意境
3. **细节**: 添加服饰、配饰、表情等细节
4. **氛围**: 通过光线、天气营造氛围
