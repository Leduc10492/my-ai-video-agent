# McKee Shared Reference Index

This folder is a shared reference index for the McKee workflow skills. It is not a standalone production workflow.

Load only the specific reference needed:

- `core/principles.md`: core story principles.
- `core/story-spine.md`: story spine and structure reminders.
- `analysis/checklists.md`: audit and diagnosis checklists.
- `analysis/mckee-cards.md`: principle cards for diagnosis.
- `analysis/mckee-workflows.md`: common McKee workflow patterns.
- `analysis/scene-analysis.md`: scene-level analysis notes.
- `coordinator/workflow-guide.md`: routing guide for McKee tasks.
- `pacing/*.md`: exposition, pacing, suspense, and irony notes.
- `source/*.md`: source map and excerpts.
- `templates/*.md`: audit, rewrite, scene-doctor, and variation templates.

- `mckee-coordinator`
- `mckee-create`
