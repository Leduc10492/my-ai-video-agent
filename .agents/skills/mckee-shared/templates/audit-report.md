# Audit Report Template

## Format

```
Title:
Medium/Format:
Act Model:
Logline:

Top Issues (ranked):
1) Issue:
   Evidence:
   Impact:
   Severity: P0/P1/P2/P3
   Fix:
2) Issue:
   Evidence:
   Impact:
   Severity: P0/P1/P2/P3
   Fix:

Story Spine Check:
- Inciting incident:
- Object of desire:
- Obligatory scene:
- Crisis choice:
- Climax:
- Resolution:

Act/Sequence Map:
- Act/Seq 1 turn:
- Act/Seq 2 turn:
- Act/Seq 3 turn:
- Act/Seq 4 turn (if 8-seq):
- Act/Seq 5 turn (if 8-seq):
- Act/Seq 6 turn (if 8-seq):
- Act/Seq 7 turn (if 8-seq):
- Act/Seq 8 turn (if 8-seq):

Scene Highlights (worst 3):
- Scene:
  Problem:
  Fix:
```

## Severity Scale

- **P0**: Story spine broken (missing inciting incident or climax)
- **P1**: Act or sequence turns weak or absent
- **P2**: Scene turns weak; value change unclear
- **P3**: Exposition or dialogue problems

## Usage

Fill out each section based on the audit findings. Focus on structural issues first (P0/P1), then scene-level issues (P2), then polish concerns (P3).
