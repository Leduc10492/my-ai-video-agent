# 场景修复表模板

Default rendered diagnosis language: Simplified Chinese unless the user explicitly requests another language. Keep scene IDs, file paths, artifact IDs, and screenplay labels unchanged when useful.

## Format

```
场景ID:
场景目标:
主角欲望:
阻力:
策略:
开场价值:
收场价值:
转折（如何翻转）:
后果（下一个问题）:

建议修改:
- [具体修复1]
- [具体修复2]
```

## Field Descriptions

- **Scene ID**: Name or number identifying the scene
- **Scene goal**: What is the scene trying to accomplish in the story?
- **Protagonist desire**: What does the character want NOW in this scene? (specific and achievable)
- **Opposition**: Who/what blocks the desire, and how?
- **Tactics**: How does the protagonist try to win?
- **Value at start**: The value state at scene opening (e.g., Hope+, Freedom-, Trust+)
- **Value at end**: The value state at scene closing (should be different from start)
- **Turn**: The moment/choice that reverses the value
- **Aftermath**: What new problem or pressure does this scene create?

## Common Problems and Fixes

| Problem | Fix |
|---------|-----|
| Character talks about desire instead of pursuing it | Replace dialogue with action that tests opposition |
| Conflict is verbal-only with no stakes | Add consequence - what does character lose if they fail? |
| Scene ends where it began (no turn) | Identify the value at stake and force it to reverse |
| Opposition gives up too easily | Give antagonist stronger motive or capability |
| Turn caused by coincidence | Turn must come from character choice under pressure |

## Usage

Fill out each field before recommending changes. If any field is unclear or missing, that's likely the source of the scene's problem. Render the filled diagnosis in Chinese by default.
