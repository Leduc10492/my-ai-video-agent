# 变化方案生成模板

Default rendered variations language: Simplified Chinese unless the user explicitly requests another language. Keep file paths, artifact IDs, and fixed workflow tokens unchanged.

## Format

```
固定锚点:
- 激励事件（固定）: [描述]
- 必有场景（固定）: [描述]
- 核心欲望（固定）: [描述]

方案A:
- 新冲突引擎: [改变了什么]
- 新中点或主要转折: [如何变化]
- 新危机选择: [主角面对什么]
- 新高潮: [冲突如何解决]
- 因果检查: [激励事件如何导致高潮]
```

## Variation Types

### Antagonist Swap
- Human → Nature
- External → Internal
- Villain → Well-meaning obstacle
- Single → Multiple distributed

### Value Swap
- Survival → Dignity
- Love → Justice
- Wealth → Freedom
- External → Internal

### Constraint Swap
- Time pressure → Resource scarcity
- Physical obstacle → Social rule
- Knowledge gap → Moral dilemma

### Tone Swap
- Tragedy → Dark comedy
- Thriller → Mystery
- Action → Psychological

## Verification Checklist

For each variation, verify:
- [ ] Inciting incident still causes climax
- [ ] Value change is clear and meaningful
- [ ] Crisis tests the STRONGEST opposing force
- [ ] Climax is irreversible

## Usage

Fix the anchor principles first, then generate 2-3 variations by changing the conflict engine. Always verify causality for each variation. Render the filled variations in Chinese by default.
