# McKee Workflows

## Audit Workflow (Story to Scene)

1. Intake
   - Identify medium, length, and format (outline, script, prose).
   - Determine current act or sequence model.
2. Story spine
   - Pin down inciting incident, crisis, climax, resolution.
   - Identify the obligatory scene promised by the premise.
3. Act or sequence map
   - Mark each major reversal and value shift.
4. Scene pass
   - For each scene: desire, opposition, turn, value change.
5. Causality pass
   - Trace cause-effect chains for each turning point.
6. Exposition pass
   - Flag any info not motivated by conflict.
7. Output
   - List issues with severity and recommended fixes.

## Rewrite Loop

1. Diagnose root cause
   - Structure problem, scene problem, or line-level problem.
2. Design fix
   - Choose structural rework, scene replacement, or line edit.
3. Re-outline
   - Update the spine and sequence map.
4. Rewrite
   - Implement changes in order of priority.
5. Re-verify
   - Check value turns, causality, and stakes.

## Scene Doctor Checklist

- Desire: What does the protagonist want in this scene?
- Opposition: Who or what blocks them?
- Tactics: How does the protagonist try to win?
- Turn: What value reverses by the end?
- Aftermath: What new problem does the scene create?

## Variation Generator

1. Fix the principle
   - Keep the inciting incident and promised payoff.
2. Change the conflict engine
   - Swap antagonist, constraint, or value at stake.
3. Preserve causality
   - Every new event must cause a later event.
4. Rebuild the climax
   - Match the new engine with a new irreversible decision.

## Severity Scale

- P0: Story spine broken (missing inciting incident or climax).
- P1: Act or sequence turns weak or absent.
- P2: Scene turns weak; value change unclear.
- P3: Exposition or dialogue problems.

## Decision Rules (Act Model)

- If length < 15 minutes: use 3-act light.
- If feature or longform: prefer 3-act with 8-sequence substructure.
- If episodic: use 4-6 sequence beats per episode, plus season arc.
