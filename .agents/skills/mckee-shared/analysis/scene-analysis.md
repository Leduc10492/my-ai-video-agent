# Scene analysis (five steps)

1. Identify the conflict driver and opposing forces.
2. Mark the opening value state (positive or negative).
3. Break the scene into beats (action/reaction exchanges).
4. Mark the ending value state and compare to the opening.
5. Locate the turning point where the value flips.

Notes:

- Beats are exchanges of action and reaction; a new beat begins when behavior changes.
- If opening and ending values are identical, the scene likely lacks meaningful change.
