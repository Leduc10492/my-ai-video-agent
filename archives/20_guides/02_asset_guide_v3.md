## 角色 1：铠冢霙（Yoroizuka Mizore）

### 参考图

![霙参考图](refs/mizore_reference.png)

### 基础信息

| 属性 | 描述 |
|------|------|
| **角色定位** | 主角，第一人称叙事者 |
| **年龄** | 高中生（约16-17岁） |
| **性格关键词** | 内敛、安静、向内、依附型、观察者 |
| **象征物** | 随身听、升学调查表、双簧管 |

### 发型规范

| 属性 | 描述 |
|------|------|
| **发色** | 黑色，带微弱蓝色反光（室内冷光下） |
| **发型** | 长直发，及腰长度（约70cm） |
| **刘海** | �的齐刘海，遮住眉毛，刚好露出眼睛 |
| **发质** | 柔顺有光泽，无卷曲 |
| **分区** | 无明显分区，自然垂落 |

**提示词关键词：**
```
long straight black hair, blunt bangs covering eyebrows, waist-length hair, silky texture, subtle blue highlights in cold light
```

### 面部特征

| 属性 | 描述 |
|------|------|
| **瞳色** | 淡褐色/灰褐色（hazel gray） |
| **眼型** | 杏眼，眼尾略下垂，带忧郁感 |
| **眉型** | 自然眉，被刘海遮挡 |
| **鼻型** | 小巧挺直 |
| **唇型** | 薄唇，自然淡粉色 |
| **肤色** | 白皙，冷白皮 |
| **面部气质** | 含蓄、内敛、略带忧郁、不易读懂 |

**提示词关键词：**
```
pale complexion, hazel gray eyes, melancholic expression, subtle emotions, introspective gaze, delicate features, slight downturned eyes
```

### 表情规范

| 情绪状态 | 表现方式 |
|----------|----------|
| **平静** | 眼神略空，嘴角平直，面部放松 |
| **期待** | 眼神聚焦，嘴角微微上扬，但克制 |
| **失落** | 眼神下垂，嘴角轻微下沉，不流泪 |
| **认清** | 眼神清澈但空洞，表情释然，不悲不喜 |
| **决定** | 眼神坚定，嘴角平直，带微弱自信 |

**重要提示：** 霙的情绪表达始终是克制的，不会大哭大笑。

## 角色 2：伞木希美（Kasakigi Nozomi）

### 参考图

![希美参考图](refs/nozomi_reference.png)

### 基础信息

| 属性 | 描述 |
|------|------|
| **角色定位** | 重要配角，霙的情感对象 |
| **年龄** | 高中生 → 成年（序幕） |
| **性格关键词** | 向外、自然、前进型、轻快、不经意 |
| **象征物** | 长笛（回忆）、iPhone（序幕）、Chanel 19手袋（序幕） |

### 发型规范

| 属性 | 描述 |
|------|------|
| **发色** | 黑色，带自然棕色光泽 |
| **发型** | 高马尾，活力感 |
| **刘海** | 空气刘海/碎刘海，露出眉毛 |
| **发质** | 柔顺自然，略有蓬松感 |
| **马尾位置** | 头顶偏后，高扎 |

**提示词关键词：**
```
black hair in high ponytail, wispy bangs, natural brown highlights, lively hairstyle, bouncy ponytail
```

### 面部特征

| 属性 | 描述 |
|------|------|
| **瞳色** | 棕色/琥珀色（amber brown），比霙更暖 |
| **眼型** | 杏眼，眼尾略上扬，有神采 |
| **眉型** | 自然眉，可见（刘海不遮挡） |
| **鼻型** | 小巧挺直 |
| **唇型** | 饱满，自然粉色，常带微笑 |
| **肤色** | 自然肤色，略带暖调，有淡淡雀斑 |
| **面部气质** | 开朗、自然、亲和、向外、不经意的魅力 |

**提示词关键词：**
```
warm complexion, amber brown eyes, bright expression, natural smile, light freckles, approachable, lively gaze, cheerful demeanor
```

### 表情规范

| 情绪状态 | 表现方式 |
|----------|----------|
| **日常** | 自然微笑，眼神明亮，轻松自在 |
| **忙碌** | 专注但不紧绷，嘴角仍有弧度 |
| **随口说话** | 轻松自然，眼神不一定聚焦对方 |
| **回头瞬间（序幕）** | 透明的确认，不是留恋，而是告别过去 |

**重要提示：** 希美的情绪表达是自然外放的，与霙形成对比。

### 服装规范

#### 校服（回忆段落）

| 部件 | 描述 | 色号参考 |
|------|------|----------|
| **上衣** | 白色水手服，短袖 | #FFFFFF |
| **领子** | 浅蓝色大翻领，白色边线 | #A8D8EA |
| **领巾** | 青蓝色/蓝绿色领巾，比霙的更深 | #4A90A4 |

**提示词关键词：**
