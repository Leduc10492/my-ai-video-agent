# BIPOLAR MV 视觉风格指南

## 目标

本指南补齐美术和视频阶段所需的统一风格来源。它从剧本、资产指南和分镜表中抽取跨镜头必须保持一致的视觉规则，供 `04_storyboard_prompts`、`05_art_prompts` 和 `06_video_prompts` 使用。

## 总体风格

- 类型：克制的日式校园 MV，带轻微胶片颗粒和现实主义质感。
- 情绪：安静、疏离、内敛，不做外放煽情。
- 画面语言：通过位置关系、道具状态、色温变化和等待动作表达关系变化。
- 表演方向：微表情优先，避免夸张哭泣、喊叫、戏剧化崩溃。

## 构图规则

- 霙与希美同框时，希美通常在前景/左侧/领先位置，霙在后景/右侧/跟随位置。
- 霙独处时，可以置于画面边缘或被窗框、桌面、书包等物体压缩空间。
- 终章中霙逐渐回到画面中心，表达主体性恢复。
- 机场序幕强调建筑线条、移动通道、登机桥、机舱窗框等冷硬结构。
- 校园回忆段强调走廊、练习室、窗台和桌面等可重复出现的关系空间。

## 光线与质感

- 序幕：冷白、金属、玻璃、雪、皮革链条的硬质感。
- 第一幕：自然光从暖转冷，避免突然强反差。
- 第二幕：柔光、暖色、轻微过曝边缘，但保留不对等构图。
- 第三幕：烟火、手机屏幕、路灯、走廊阴影形成冷暖切换。
- 终章：清晨或室内柔光，干净、明亮但不梦幻化。

## 道具风格

- 随身听：银灰金属、2000年代随身设备、机械磁带窗口可见。
- 升学调查表：A4白色表格纸，空白 → 折痕 → 摊开 → 填写完成。
- iPhone/耳机/Chanel 19：仅用于成年希美序幕，作为向外、现代、物质世界符号。
- 乐器：霙为双簧管，希美为长笛，避免混用。

## Prompt 使用规则

- 生成线稿分镜时：使用 black and white storyboard sketch / graphite line art / panel borders / technical notes。
- 生成最终美术图时：使用 cinematic still / soft film grain / restrained expression / consistent character DNA。
- 生成视频提示词时：运动必须服务叙事，不为炫技添加无意义镜头运动。
- 每个涉及角色的 prompt 必须引用资产指南中的角色 DNA，而不是使用泛称。
