# Shotlist Scene Packages

每个 screenplay Scene 使用一个直接目录：

```text
scene-021_v1/
├── 03_shotlist_breakdown_scene-021_v1.md
├── Shotlist_scene-021_ZH_v1.html
├── manifest.md
├── spatial_blocking.md        # 需要时
├── assets/                    # 仅 Scene 专用资产
├── previews/                  # 用户单独授权后
└── generated/                 # 用户单独授权后
```

生产层级保持 `Scene -> Shot Row -> Prompt Envelope`；只为 Scene 建目录。
